## To Run the Code

1. Clone the repository:

```bash
git clone
```

2. Navigate to the project directory:

```bash
cd javascript-solutions
```

3. Install the dependencies:

```bash
npm install
```

4. Play around, test the functions, and explore the code!

5. Run the tests:

```bash
npm test
```
